# Notify - https://notify-notes.vercel.app/

## About
A Note Taking App

## Tech Stack
- ReactJs
- JavaScript
- Backend - Mockbee(https://mockbee.netlify.app/)

## Features 
- User Authentication
- Archive
- Trash
- Note Label Management
- Note Color and styling
- Pin a note
- Filter notes
